# Plantilla de reac

## Available Scripts

Proyecto de react con configuraciones bases:
-React router dom
-React redux
-Alias path
-Cache context

Ejecutar en la raiz del proyecto:

### `yarn dev`

Para correr la app en modo de desarrollo.\
Abrir [http://localhost:3000](http://localhost:3000) para ver en el navegador.

### `yarn test`

### `yarn build`
